# Specification

## Summary
**Goal:** Rename the chat section to "Chat with Nobita", upgrade the chatbot with an intelligent rule-based Nobita persona, and add three new Doraemon-themed mini-games.

**Planned changes:**
- Rename all instances of "Chat with me" to "Chat with Nobita" across navigation, page headings, dashboard portal card, and route titles
- Replace static bot responses with a rule-based conversational AI engine using Nobita's persona — covering greetings, factual questions, gadgets, jokes, emotions, school/homework, friendship, food, weather, and more with varied, funny, in-character replies
- Add three new mini-games to GamesPage with their own routes:
  1. **Doraemon Gadget Clicker** — click gadgets before time runs out
  2. **Nobita's Exam Quiz** — funny multiple-choice Doraemon trivia quiz
  3. **Whack-a-Doraemon** — whack-a-mole style game with Doraemon characters
- Each new game includes a score display, restart button, and Doraemon-themed UI
- Register all new game routes in App.tsx

**User-visible outcome:** Users can chat with a funny, in-character Nobita bot that responds meaningfully to a wide range of topics, and can play three new entertaining Doraemon-themed mini-games from the Games page.
